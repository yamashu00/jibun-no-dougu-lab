## 1. このプロジェクトで実現したいこと

> **「ある一人の学生のために、その人だけの声を作る」**

大学の支援室には、コミュニケーションに困難を抱える学生がいる。
授業で質問したくても声が出ない。食堂でうまく注文できない。困っているのに助けを求められない。

市販のAACデバイスや支援ツールは「平均的な障がい像」をもとに設計されており、個々の学生が実際に使う場面・言葉・操作感には合っていないことが多い。

このプロジェクトは、**一人ひとりの学生と丁寧に対話しながら、その人専用のコミュニケーション補助デバイスを高専生が設計・製作する**というものです。
作ったら終わりではなく、使ってもらい、フィードバックをもらい、直す。
そのやりとりの積み重ねの中から、本当に必要なものが生まれます。

## 3. プロジェクトの核心：「やりとり」の設計

このプロジェクトが大切にしているのは、**作ること** ではなく **やりとりのプロセス** です。

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  当事者学生  ──── ニーズを語る ────▶  高専生チーム  │
│            ◀─── 試作品を届ける ────                 │
│                                                     │
│  当事者学生  ──── 使ってみる ──────▶  高専生チーム  │
│            ◀─── フィードバック ────                 │
│                                                     │
│  当事者学生  ──── 直してみる ──────▶  高専生チーム  │
│            ◀──── また試す ─────────                 │
│                                                     │
│               ↑ このサイクルを繰り返す               │
│                                                     │
└─────────────────────────────────────────────────────┘
```

高専生が経験するのは「誰かのために作るものづくり」です。
仕様書が先にあるのではなく、人との対話から仕様が生まれる。
これは、神山まるごと高専が目指すデザイン×エンジニアリングの本質的な実践です。

## 4. 作るデバイス：「声のかわり」

### 基本コンセプト

手のひらサイズの物理デバイス。ボタンを押すと、登録したフレーズを音声で読み上げる。
**すべてがオーダーメイド** — ボタンの数も配置も、言葉も声のトーンも、その人に合わせて作る。

### デバイスのイメージ

```
┌──────────────────────────────────┐
│                                  │
│   ┌──────┐  ┌──────┐  ┌──────┐  │
│   │  ✅  │  │  🙅  │  │  🙋  │  │
│   │ はい │  │いいえ│  │質問が│  │  ← 場面ごとのボタン
│   └──────┘  └──────┘  └──────┘  │    ラベル・フレーズ・
│                                  │    アイコンをカスタマイズ
│   ┌──────────────────────────┐   │
│   │   🆘  助けてください     │   │  ← 大きいボタン（緊急用）
│   └──────────────────────────┘   │
│                                  │
└──────────────────────────────────┘
         📱 スマートフォンアプリで
           フレーズ・ボタンを管理
```

### 主なカスタマイズ項目

| 項目 | 内容 |
|------|------|
| ボタンの数・配置 | 2個〜最大10個程度。手の大きさや可動域に合わせて配置を変更 |
| ボタンサイズ | 押しやすい大きさに変更（3Dプリントで成形） |
| 登録フレーズ | 授業中・食堂・支援室など場面別に登録。本人の言葉で作る |
| 音声トーン | 読み上げスピード・ピッチを調整。本人が「これだ」と思える声に |
| 緊急ボタン | 大きいボタンに「助けてください」などを割り当て |
| 筐体の形 | 手型スキャンをもとに3Dプリントで成形（上位バージョン） |

### 管理アプリ（デモ版 完成済み）

スマートフォン・PCのブラウザで動くデモアプリを作成しました。
**`app.html`** をブラウザで開くと実際に触れます。

機能：
- ボタン一覧の確認・追加・編集・削除
- フレーズの読み上げテスト（日本語音声合成）
- 場面（授業中・食堂・支援室・すべて）でのフィルタリング
- 音声スピード・ピッチ・音量の調整

## 9. この共同申請の強み

### 審査委員から見た独自性

1. **「支援専門家 × 工学生」という組み合わせ** — 2025年度採択20件にこのパターンはほぼない
2. **当事者が開発プロセスに参加する** — 既製品の購入ではなく、共同設計であること
3. **高専教育との接続** — 助成金が高専生の学習経験も変える（二重の教育効果）
4. **低コスト・再現可能** — 完成したモデルは他校でも展開できる

### 財団の趣旨との一致度

| 財団の求める要素 | 本プロジェクトでの対応 |
|----------------|----------------------|
| 障がいのある青少年の修学支援 | ✅ 郡山女子大学の障がい学生が直接受益 |
| IT技術を活用 | ✅ マイコン・3Dプリント・Webアプリ開発 |
| 研究・開発 | ✅ 共同設計プロセスの体系化・効果検証 |
