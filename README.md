# じぶんの道具ラボ 🔧

自助グッズ共同設計プラットフォーム — プロトタイプ

CTC未来財団 研究・開発助成 申請プロジェクト  
郡山女子大学 稲田健実 × 神山まるごと高専 山本周

## デモ

👉 **https://yamashu00.github.io/jibun-no-dougu-lab/**

- 「道具を探す」— 制作実績カタログ（カテゴリフィルタ付き）
- 「道具を依頼する」— テキストで困りごとを送信 → 進捗タイムライン

## コンセプト

困りごとをテキストで伝えるだけで、高専生チームが自助グッズを設計・製作。  
使ってフィードバックし、一緒に改善するプロセスを通じて、  
「自分のニーズを言語化する力」（就労支援につながる自己開示力）を育てる。
